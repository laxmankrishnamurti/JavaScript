# Day - 04 Loops

## Task/Activity

- Activity 01: For loop

  - Task 01: Write a program to print numbers from 1 to 10 using for loop
  - Task 02: Write a program to print the multiplication table of 5 using a for loop.

- Activity 02: While Loop

  - Task 03: Write a program to calculate the sum of numbers from 1 to 10 using a while loop.
  - Task 04: Write a program to print numbers from 10 to 1 using a while loop.

- Activity 03: D0...While Loop

  - Task 05: Write a program to print numbers from 1 to 5 using a do...while loop.
  - Task 06: Write a program to calculate the factorial of a number using a do...while loop.

- Activity 04: Nested Loops

  - Task 07: Write a program to print a pattern using for loops

    - Pattern ::

    ```javascript
        *
        * *
        * * *
        * * * *
        * * * * *
        * * * * * *
        * * * * * * *
    ```

- Activity 05: Loop Control Statements
  - Task 08: Write a program to print numbers from 1 to 10. but skip the number 5 using the <code>continue</code> statement.
  - Task 09: Write a program to print numbers from 1 to 10, but stop the loop when the number is 7 using the <code>break</code> statement.
